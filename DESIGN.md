# NAS Music Player - Symfonium 风格音乐播放器

## 产品定位

一个专为 NAS 音乐收藏设计的高端音乐播放器，融合赛博朋克美学与专业音频控制体验。

## 设计风格：暗黑科技风 (Cyber Tech)

### 视觉关键词
- 纯黑沉浸背景
- 霓虹发光效果
- HUD 界面质感
- 未来感交互

### 配色方案
| 用途 | 色值 | 名称 |
|------|------|------|
| 主色 | #00F0FF | 电光青 |
| 辅色 | #BF00FF | 霓虹紫 |
| 背景 | #0A0A0F | 纯黑 |
| 卡片 | #12121A | 微亮黑 |
| 文字主 | #EAEAEA | 亮白 |
| 文字副 | #555570 | 暗灰 |
| 强调 | #00FF88 | 矩阵绿 |

### 核心样式手法
1. **霓虹发光边框**：`borderColor: rgba(0,240,255,0.3)` + `shadowColor: #00F0FF`
2. **小圆角**：卡片 `borderRadius: 8`，按钮 `borderRadius: 6`
3. **渐变发光条**：LinearGradient 从电光青到霓虹紫
4. **等宽字体**：数字使用 JetBrains Mono 风格

## 页面结构

### 导航设计（底部 Tab Bar）
- **首页**：最近播放 + 推荐内容
- **库**：艺术家/专辑/歌曲分类浏览
- **搜索**：全局音乐搜索
- **列表**：播放队列管理
- **设置**：NAS 连接配置

### 核心页面

#### 1. 首页 (Home)
- Hero 区域：当前播放歌曲封面 + 歌曲信息
- 最近播放列表
- 推荐专辑网格

#### 2. 音乐库 (Library)
- 顶部切换：艺术家 / 专辑 / 歌曲
- 瀑布流/列表展示
- 点击进入详情页

#### 3. 专辑详情
- 大封面展示
- 歌曲列表
- 播放全部/随机播放按钮

#### 4. 播放页面
- 全屏封面
- 进度条 + 时间显示
- 播放控制（上一首/播放暂停/下一首）
- 音量控制
- 播放模式切换

#### 5. 迷你播放器
- 底部常驻
- 封面缩略图 + 歌曲名 + 播放/暂停按钮
- 点击展开到播放页面

## 功能清单

### 核心功能
- [ ] NAS 连接配置与保存
- [ ] 音乐库同步与浏览（艺术家/专辑/歌曲）
- [ ] 音乐播放（流媒体播放）
- [ ] 播放控制（播放/暂停/上一首/下一首/进度）
- [ ] 播放队列管理
- [ ] 歌曲搜索
- [ ] 专辑封面显示
- [ ] 播放模式（顺序/随机/单曲循环）

### 技术实现
- 前端：Expo + expo-av 音频播放
- 后端：Express.js 代理 NAS 文件访问
- 状态管理：React Context

## 设计禁忌
- 避免使用白色/亮色背景
- 禁止使用圆润过度的设计
- 禁止使用传统 iOS 风格图标
